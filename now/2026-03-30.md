## Where Am I?

Living on the west side of the Portland, Oregon metropolitan area.

## What Am I Working On?

The following are projects, tasks or hobbies that I am currently working on:

* Decommissioning my personal Pixelfed instance
  * I cannot justify the cost of running my own instance that I use to share photos of the notes that I take while listening to [Wait Wait... Don't Tell Me!](https://waitwait.npr.org/) and events that I attend
  * For those photos, I upload the full set of photos to [my Flickr account](https://www.flickr.com/questionlp)
  * In preparation of decommissioning my Pixelfed instance, I found some significant blockers when trying to migrate the account to [my personal Mastodon instance (linh.social)](https://linh.social/)
  * There is also no clean way to export the posts (or statuses as Pixelfed call them) along with the images

## What Would I Like To Do Next?

I have a couple of things that I would like to spend time doing or trying out, in no particular order:

* Stand up a private Forgejo instance as an option to migrate private repositories hosted on GitHub
* Start migrating some of my project repositories from GitHub to Codeberg
* Researching ORMs (object relational mappers) to use for the next major versions of the Wait Wait Stats Project applications and libraries
* Researching the process to potentially migrate the Wait Wait Stats Project from MySQL or MariaDB to PostgreSQL
* Determine a use for my Microsoft Surface Pro 3, which has been collecting dust
* Replace the two Proxmox hosts running at home with something newer and more power efficient
