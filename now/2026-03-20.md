## Where Am I?

Living on the west side of the Portland, Oregon metropolitan area.

## What Am I Working On?

The following are projects, tasks or hobbies that I am currently working on:

* Switching from [Alacritty](https://alacritty.org/) to [Ghostty](https://ghostty.org/) as my primary terminal application
  * I wanted to try out the native split view functionality without having to use screen or tmux
  * Enabling the scrollbar to always be displayed in GNOME causes a rendering problem where the scrollbar background is fully transparent
  * Due to a bug with GTK, the window height and width value needs to be adjusted to account for the titlebar and other window decorations. In my case, I had to increase the window height by 3 characters to match the working dimensions set in Alacritty
* Trying out [Fedora Silverblue](https://fedoraproject.org/atomic-desktops/silverblue/) on my ThinkPad T14 Gen 2 AMD to see what would be required to change my development workflow, which does not make use of containerization.
  * [toolbx](https://containertoolbx.org/) is a nice and easy way to install applications within containerization environments via Podman
  * Need to look at how to run MariaDB and PostgreSQL database servers within Podman (or LXC) containers
